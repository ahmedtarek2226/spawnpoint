## 🚀 What is spawnpoint?

spawnpoint is a self-hosted dashboard for managing Minecraft servers from one place. It helps you start, stop, and monitor servers with a simple web interface. You can use it on Windows to manage a local setup or a home server.

It is built for people who want control without dealing with a long list of command prompts. You open the dashboard in your browser, sign in if needed, and manage your server from there.

## 🔧 Managing your server

From the dashboard, you can handle common tasks without opening the server folder each time.

### Start and stop
Use the main controls to start or stop a server. This helps when you need to restart after a change or update.

### Check logs
Logs show what the server is doing. If a mod fails to load or a world does not start, the logs help you find the cause.

### Backups
Keep backups before large changes. This protects your world if something goes wrong during an update or mod install.

### Server files
You can manage files from the dashboard in a more direct way. That makes it easier to update configs, add mods, or change server options.

### Status view
See if the server is online, how much memory it uses, and whether players are connected. This helps you spot issues fast.

## ❓ If the server will not start

If your Minecraft server does not start, check these points:

- Make sure Java is installed
- Confirm the server files are in the right folder
- Check the logs for errors
- Make sure the port is free
- Confirm Docker is running if you use Docker
- Check that the server jar or mod files match your setup
- Make sure your world files are not locked by another app

Many startup problems come from a wrong file path or a missing Java version.

## 📌 File and folder tips

Keep your server files in a folder that is easy to find. A simple path like `C:\MinecraftServers\MyServer` helps you avoid confusion. Do not move files while the server is running.

If you use backups, store them in a separate folder. That makes restores easier.
